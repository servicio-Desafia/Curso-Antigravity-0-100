# Guía de Configuración del Entorno - Pime IA

Bienvenido al "Taller" de Pime IA. Antes de construir, necesitamos preparar nuestras herramientas.

## 1. El Motor: Python
Python es el cerebro que procesará nuestros datos.

1.  Descarga Python (versión 3.10 o superior) desde [python.org](https://www.python.org/downloads/).
2.  **IMPORTANTE:** Al instalar, marca la casilla que dice **"Add Python to PATH"**. Esto es vital.

## 2. La Mesa de Trabajo: VS Code
Visual Studio Code es donde escribiremos nuestras instrucciones.

1.  Descarga VS Code desde [code.visualstudio.com](https://code.visualstudio.com/).
2.  Instálalo con las opciones por defecto.
3.  Abre VS Code e instala la extensión **"Python"** (busca el ícono de los cuadritos a la izquierda).

## 3. El Bitácora: Git (Opcional por ahora)
Git nos permite guardar versiones de nuestro trabajo. Lo configuraremos más adelante, pero si quieres adelantar:
[Descargar Git](https://git-scm.com/downloads)

## 4. Probando que todo funciona
1.  Abre VS Code.
2.  Abre la terminal (`Ctrl + ñ` o menú Terminal > New Terminal).
3.  Escribe: `python --version`
4.  Si ves algo como `Python 3.12.x`, ¡estamos listos!
